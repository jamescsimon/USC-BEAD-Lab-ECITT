# Copy sprite images from existing ECITT_Web_App

Copy these files from: `codebases/ECITT_Web_App/public/graphics/sprites/`

To: `ipad-app/public/sprites/`

Files needed:
- bfly-01.png through bfly-08.png
- bubble-01.png, bubble-02.png
- frog-01.png, frog-02.png, frog-03.png
- snowflake-01.png, snowflake-02.png

Or run:
```
xcopy ..\..\codebases\ECITT_Web_App\public\graphics\sprites\* public\sprites\ /Y
```
