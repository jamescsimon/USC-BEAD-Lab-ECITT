# Controller Workflow
1. go to localhost:8000
2. [Controller]
3. [Connect]
4. [Tests]
5. [Add new] project → Save → Enter
6. [Add new] test set → Save → Enter
7. [Add new] participant → fill ref/birth/gender → Save → Enter
8. Select "Adult" test → Enter
9. Run trials:
   - Test PR Top: [x 4] Prtc PR Top → [x 32] Test PR Top
   - Test PR Btm: [x 4] Prtc PR Btm → [x 32] Test PR Btm
   - Control: [x 4] Ctrl PR Btm → [x 4] Ctrl PR Top → [x 4] Ctrl PR Mdl

# Participant Workflow
1. go to ngrok url (or http://YOUR_IP:8000)
2. [Responder]
3. [Connect]
4. Wait for "Available for Connection" or "Ready"
5. Trials appear automatically when Controller starts them
